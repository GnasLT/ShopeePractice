1. Cloud & CDN Services (Dịch vụ đám mây & Phân phối nội dung)
Cloud Providers (AWS / GCP):

AWS Compute & Containers: EC2, AWS Lambda (Serverless), ECS / EKS (Kubernetes).

AWS Storage & Databases: S3 (Object Storage), DynamoDB (NoSQL), RDS (PostgreSQL/MySQL), ElastiCache (Redis).

AWS Networking & Security: VPC, Route 53 (DNS), ALB/NLB (Load Balancers), IAM, AWS WAF.

AWS Messaging: SQS (Message Queue), SNS (Pub/Sub), MSK (Managed Kafka).

CDN & Edge Computing:

Cloudflare / AWS CloudFront: Cấu hình Cache Rules, Edge Workers (Cloudflare Workers / CloudFront Functions), WAF (chống DDoS/XSS/SQLi ở tầng L7), SSL/TLS Optimization.

2. Distributed Systems & High Concurrency (Hệ thống phân tán & Tải cao)
Message Brokers & Event Streaming: Apache Kafka, RabbitMQ, NATS.

Caching & In-Memory Databases: Redis (Redis Cluster, Sentinel, Redlock), Memcached.

Concurrency & Rate Limiting: Distributed Locks, Token Bucket / Leaky Bucket Algorithms, API Throttling.

Data Consistency: Distributed Transactions, Saga Pattern, Outbox Pattern, CRDTs (Conflict-free Replicated Data Types).

3. Database & Storage Architecture (Cơ sở dữ liệu & Lưu trữ)
Relational DB (RDBMS): PostgreSQL, MySQL (Kỹ thuật: Indexing, Read/Write Splitting, Partitioning, Sharding, Query Optimization).

NoSQL & Time-Series DB: MongoDB, Cassandra, ClickHouse, Prometheus (Time-Series).

Search Engine: Elasticsearch / OpenSearch (Full-text search, Logging).

Storage Mechanics: Write-Ahead Logging (WAL), Append-Only File (AOF).

4. Frontend & Web Real-time (Công nghệ Client & Thời gian thực)
Core Frontend & Frameworks: JavaScript (ES6+), TypeScript, React.js / Next.js, Vue.js / Nuxt.js.

Web Performance Optimization: Core Web Vitals (LCP, INP, CLS), Code-Splitting, Tree-Shaking, Lazy Loading, Service Workers (PWA).

Rendering Strategies: SSR (Server-Side Rendering), SSG (Static Site Generation), ISR, Micro-Frontends.

Real-time & High-FPS Protocols: WebSockets, Server-Sent Events (SSE), WebRTC (P2P), HTML5 Canvas, WebGL.

5. Security & Authentication (Bảo mật & Xác thực)
Auth Protocols: OAuth 2.0, OpenID Connect, JWT (JSON Web Token), Session Management, RBAC/ABAC.

Web Security Standards: OWASP Top 10, Content Security Policy (CSP), CORS, CSRF Protection, Input Sanitization (chống XSS/SQLi).

6. DevOps, Infrastructure & Observability (Hạ tầng & Tự động hóa)
Containerization & Orchestration: Docker, Docker Compose, Kubernetes (K8s).

Infrastructure as Code (IaC): Terraform, Ansible.

CI/CD Automation: GitHub Actions, GitLab CI/CD, Jenkins.

Observability & Monitoring (PLG/ELK Stack): Grafana, Prometheus, Elastic Stack, Jaeger / OpenTelemetry (Distributed Tracing).

Testing & Stress Testing: k6, Locust, Jest, Playwright / Cypress.
